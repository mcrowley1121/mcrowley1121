### 🛠️ What I do

I work on the platform layer of a global org, where my day-to-day is roughly:

- **Infrastructure as Code** — Azure environments defined and versioned in Terraform, deployed through GitHub Actions.
- **Databricks platform engineering** — Unity Catalog governance, catalog/schema/table/volume creation, secret scopes, budget policies, connecting data sources, and onboarding other data teams.
- **FinOps & cost attribution** — making cloud spend legible and chargeable back to the teams that incur it.
- **Security & identity** — CVE remediation in container images, OIDC federated credentials, and RBAC and ABAC access automation.

I like to automate things and make people's lives easier.
